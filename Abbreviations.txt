Error = Standard error
MCS = Monte-Carlo Step
Neigh = Proportion of aggregate cells
Nombre = Name of Particle condition
Numero_ag = Number of aggregates
Parametro = Value of parameter 
Particula = Particle condition (size x 1/density)
Replica = Replica

Tipo = Type of Size : Mean (Promedio) or Maximum (Maximo)
